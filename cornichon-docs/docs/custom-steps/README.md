# Custom steps

When the built-in steps of Cornichon are not enough for what you want to express, it is time to write your own custom steps.

There are currently 4 kinds of custom steps available

- [EffectStep](effect-step.md)
- [AssertStep](assert-step.md)
- [ScenarioResourceStep](resource-step.md)
- `WrapperStep` — see the [Wrapper Steps](../dsl/wrapper-steps.md) section in the DSL reference
